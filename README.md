<h1 align="center">One-step-admin</h1>

<p align="center">一款具备<b>全新交互方式</b>的 Vue 中后台管理系统框架</p>

## 特点

- 高效的交互方式，让使用人员可以跨模块的多线操作
- 提供主题配置文件，轻松实现个性化定制
- 精心设计的动效，让每一处的动画都干净利落
- 根据配置自动生成导航栏
- 支持全方位权限验证
