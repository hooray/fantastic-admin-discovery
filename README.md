## 特点

- 高效的交互方式，让使用人员可以跨模块的多线操作
- 提供主题配置文件，轻松实现个性化定制
- 精心设计的动效，让每一处的动画都干净利落
- 根据配置自动生成导航栏
- 支持全方位权限验证

## 分支说明

> 基础版可免费下载用于商业使用。
> 
> 建议刚接触的开发者，可以在本地同时下载并运行演示分支和模版分支，模版分支用于项目开发，演示分支可做为示例参考。

- `master` 演示分支，同演示站
- `template` 模版分支，移除演示代码，可直接用于实际开发
