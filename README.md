## 特点

- 可自由替换 UI 组件库，默认使用 Element Plus
- 高效的交互方式，让使用人员可以跨模块的多线操作
- 丰富的布局与主题
- 提供系统配置文件，轻松实现个性化定制
- 根据配置自动生成导航栏
- 支持全方位权限验证
- 轻松实现国际化多语言适配

## 分支说明

- `main` 框架源码分支，不含示例代码，可直接用于实际开发
- `example` 演示源码分支，同演示站，包含大量示例，可用于参考学习
